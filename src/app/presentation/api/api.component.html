API reference for Amazing Time Picker

import {{ '{' }} AmazingTimePickerService {{ '}' }} from 'amazing-time-picker';

Directives

Name Description
atp-time-picker You can add this directive to any input element. It will read the value of element, if its there with 00:00 format, and it is compatible with HTML 5 input time.

Services

You can use amazing time picker service in order to open dialog programmatically.

AmazingTimePickerService

You can simply import this service to any component that might need to display a dialog.
Name Description
AmazingTimePickerService.open() Opens the dialog, and returns a function that you can subscribe to, so when dialog is closed, it's gonna give you the result as a Observable {{'<'}}string{{'>'}}
